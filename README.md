I build web applications that are fast, intentional, and well-tested — with a frontend focus and growing full-stack experience. When I'm not in the codebase, I'm researching molecular isotopes and thinking about what stars can tell us.

I believe the best interfaces, like the best ideas, emerge slowly and consistently.
